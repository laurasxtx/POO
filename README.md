# POO
curso de progrmacion orientada a objetos
Para poder realizar y entender mejor lo que es el ascensor y su funcionamiento, se creo el diagrama de clase donde en este se determinaron clases, métodos, atributos y herencias las cuales nos van a ayudar de mucho para crear el código. Según este diagrama es como comencé a crear el código para la simulación básica de un ascensor.
 
Primero comencé por la Declaración e Inicialización de Las variables de estado del ascensor ( capacidad, pisoActual, estado, direccion) y su componente PuertaAscensorse declaran y se les asignan valores iniciales en el constructor.
 Luego implemente la lógica de operación en donde Los métodos (subir, bajar, abrirPuerta, cerrarPuerta, detener)cambian el estado del ascensor y controlan su puerta. 
Por consiguiente esta la encapsulación en donde Se utilizan métodos getter y setter para acceder a los valores del estado del ascensor.
Y por último la ejecución y la Prueba en donde el método maindemuestra cómo crear un objeto Ascensor, como se relacionan con él llamando a sus métodos, y luego leer su estado final usando los getters.
